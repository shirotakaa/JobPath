Tentang Pekerjaan

Sebagai Frontend Developer, Anda akan berperan dalam membangun dan mengoptimalkan pengalaman pengguna melalui pengembangan antarmuka web yang responsif, interaktif, dan berkualitas tinggi.

Anda akan bekerja sama dengan tim desain dan backend untuk menciptakan produk yang fungsional, estetis, dan mudah digunakan.

Pekerjaan yang Akan Anda Lakukan

  • Mengembangkan antarmuka web yang responsif dan interaktif menggunakan HTML, CSS, dan JavaScript.
  • Bekerja sama dengan tim desain UI/UX untuk menerjemahkan wireframe dan desain ke dalam kode yang berfungsi.
  • Mengoptimalkan performa dan aksesibilitas halaman web untuk pengalaman pengguna yang lebih baik.
  • Menulis kode yang bersih, efisien, dan dapat dipelihara dengan mengikuti standar pengembangan yang baik.
  • Bekerja dengan tim backend untuk mengintegrasikan API dan memastikan pengalaman pengguna yang lancar.
  • Melakukan debugging dan troubleshooting pada tampilan serta fungsionalitas aplikasi web.
  • Mengikuti tren terbaru dalam teknologi frontend dan menerapkannya ke dalam proyek.

Keahlian yang Diperlukan

  • Pengalaman dalam pengembangan web menggunakan HTML, CSS (SCSS), dan JavaScript (ES6+).
  • Familiar dengan framework frontend seperti React, Vue, atau Angular.
  • Memahami konsep desain responsif dan mobile-first.
  • Kemampuan untuk mengoptimalkan performa website (lazy loading, caching, minification, dll.).
  • Memiliki pengalaman dalam mengembangkan dan menggunakan component design system.
  • Memahami konsep UI/UX dan bagaimana mengimplementasikannya dalam pengembangan frontend.
  • Kemampuan debugging dan troubleshooting menggunakan DevTools browser.
  • Pengalaman dengan Git dan workflow pengembangan berbasis version control.

Kualifikasi

  • Pendidikan minimal S1 di bidang Teknik Informatika, Sistem Informasi, atau bidang terkait (atau pengalaman setara).
  • Minimal 2-5 tahun pengalaman sebagai Frontend Developer atau posisi serupa.
  • Pemahaman tentang konsep REST API dan GraphQL.
  • Pengalaman dengan alat pengembangan seperti Webpack, Vite, atau Gulp.
  • Kemampuan untuk bekerja dalam tim dan berkomunikasi dengan baik.
  • Berorientasi pada detail dan memiliki kemampuan problem-solving yang kuat.
  • Keinginan untuk terus belajar dan mengikuti perkembangan teknologi frontend.
